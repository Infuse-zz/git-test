Почему код даёт именно такие результаты?

Переменные: var a = 1, b = 1, c, d;

1. c = ++a; alert(c); // 2

операция инкремент, прибавление заданного числа происходит до вывода - префиксная форма

2. d = b++; alert(d); // 1

операция инкремент, прибавление заданного числа после вывода - постфиксная форма

3. c = (2+ ++a); alert(c); // 5

к переменной a (она уже имеет значение 2) применяется операция инкремент. значение a становится 3. после этого прибавляется 2. Результат 5

4. d = (2+ b++); alert(d); // 4

к переменной b (значение 2) применяется операция инкремент (не выводится, так как постфиксная) и прибааляется 2. Результат 4.

5. alert(a); // 3

после двух операций инкремента значение a становится 3

6. alert(b); // 3

после двух операций инкремента значение b становится 3
